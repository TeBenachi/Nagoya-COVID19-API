# Nagoya-COVID19-API
Nagoya, Aichi COVID19 API

This API shows the number of confirmed COVID-19 in Nagoya, Japan. 

#### Goal:

(1) Add culster mapping  (Circle color, size)

(2) Add number of confirmed
